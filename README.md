#📱Social Spark App

## Description
The app suggests social interaction a social interaction based on the time of day.

## Features Include:
- User input for time of day
- Displays a social suggestion
- Generate social spark button
- Reset button
- Error handling for invalid input
- Error handling for empty input

## Built With
- Kotlin
- Android Studio

## GitHub Actions
A GitHub Action is optionally named and defines when it should run (e.g., on a pull request). This project uses GitHub Actions to automatically build the app when code is pushed.
